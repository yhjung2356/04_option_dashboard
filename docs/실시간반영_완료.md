## 💡 Jenkins? Tomcat?

### Jenkins
**필요 없습니다!** (중소 프로젝트는)
- GitHub Actions가 100배 쉽고 빠름
- 무료
- 설정 10분
- 유지보수 거의 없음

### Tomcat
**이미 있습니다!**
- Spring Boot = 내장 Tomcat
- 별도 설치 불필요
- JAR 파일 실행으로 끝

## 📈 생산성 향상

### Before (옛날 방식)
```
파일 수정 (1분)
→ 서버 중지 (10초)
→ 빌드 (1분)
→ 서버 시작 (30초)
→ 브라우저 확인 (10초)

총 소요 시간: 2분 50초
하루 10번 = 28분 낭비
```

### After (핫 리로드)
```
파일 수정 (1분)
→ Ctrl+S (1초)
→ F5 (1초)

총 소요 시간: 1분 2초
하루 10번 = 10분
→ 18분 절약! 🎉
```

## 🎁 보너스: GitHub Actions

### 자동 배포 설정 (선택사항)

#### 1. GitHub Secrets 설정
```
Repository → Settings → Secrets
```

필요한 Secrets:
- `DOCKER_USERNAME`: Docker Hub 사용자명
- `DOCKER_PASSWORD`: Docker Hub 비밀번호
- `EC2_HOST`: AWS EC2 IP (선택)
- `EC2_USER`: SSH 사용자 (선택)
- `EC2_SSH_KEY`: SSH 키 (선택)

#### 2. Git Push만 하면 끝!
```bash
git add .
git commit -m "기능 추가"
git push origin main

# GitHub Actions 자동 실행:
# 1. 빌드 ✅
# 2. 테스트 ✅
# 3. Docker 이미지 생성 ✅
# 4. (선택) EC2 배포 ✅
```
