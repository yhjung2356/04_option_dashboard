# 🚀 빠른 시작 - 핫 리로드 테스트

## 1분만에 시작하기!

### Step 1: 핫 리로드 실행
```bash
hot-reload.bat
```

### Step 2: 파일 수정 테스트
1. `src\main\resources\templates\dashboard.html` 열기
2. 아무 텍스트나 수정 (예: 제목 변경)
3. **Ctrl+S** 저장
4. 콘솔에서 `Restarting...` 메시지 확인
5. 브라우저 **F5** 새로고침
6. ✅ **변경사항 즉시 반영!**

### Step 3: CSS 수정 테스트
1. `src\main\resources\static\css\dashboard.css` 열기
2. 색상 변경
3. **Ctrl+S** 저장
4. 브라우저 **F5**
5. ✅ **스타일 즉시 반영!**

### Step 4: JavaScript 수정 테스트
1. `src\main\resources\static\js\dashboard.js` 열기
2. console.log 추가
3. **Ctrl+S** 저장
4. 브라우저 **F5**
5. ✅ **스크립트 즉시 반영!**

## 더 빠르게 하고 싶다면?

### LiveReload 브라우저 확장 설치 (선택)
- Chrome: LiveReload Extension
- 설치 후 활성화하면 **자동 새로고침!**
- F5 누를 필요도 없음!

## 문제 해결

### Q: "Restarting..." 메시지가 안 나와요
**A:** DevTools가 비활성화 되어있을 수 있습니다.
```bash
# 다시 실행
mvnw spring-boot:run
```

### Q: 변경사항이 반영 안 돼요
**A:** 브라우저 캐시 완전 삭제
```
Ctrl+Shift+Delete → 캐시 삭제
또는 Ctrl+F5 (강력 새로고침)
```

### Q: 속도가 느려요
**A:** IDE의 자동 빌드 기능 활성화
- Eclipse: Project → Build Automatically
- IntelliJ: Build → Build Project Automatically
