# 🎉 2025-12-21 업데이트 완료

## ✅ 완료된 작업

### 1. 💡 투자자 안내문 기능 추가

#### 변경 사항
- **저장 버튼 → 느낌표 버튼**: 기존 저장 아이콘을 느낌표 아이콘으로 교체
- **팝업 모달 추가**: 투자자 안내문을 표시하는 모달 창 구현
- **주황색 강조**: 중요성을 나타내는 주황색 버튼 스타일

#### 투자자 안내문 내용
- ⚠️ **투자 위험 고지**
  - 투자 권유나 매매 추천 목적이 아님을 명시
  
- 📋 **주요 유의사항**
  - 고위험 상품 경고
  - 레버리지 위험 안내
  - 데이터 정확성 고지
  - 투자 책임 명시

- 💡 **권장사항**
  - 충분한 학습 권장
  - 금융상품 설명서 확인
  - 여유 자금 투자
  - 전문가 상담 권유

#### 사용 방법
```
1. 헤더 오른쪽의 노란색 느낌표 버튼 클릭
2. 투자자 안내문 팝업 확인
3. X 버튼 또는 모달 외부 클릭으로 닫기
```

### 2. 🔧 시장 상태 표시 개선

#### 변경 내용
- **"주말 휴장" → "장 마감"으로 통합**
  - 기존: 주말에는 "주말 휴장", 평일 비거래시간에는 "장 마감"
  - 개선: 모든 비거래시간을 "장 마감"으로 통일

#### 이유
- 사용자에게 더 직관적인 메시지
- 중복된 상태 표시 제거
- 심플하고 일관된 인터페이스

#### 시장 상태
- ✅ **주간장 거래중**: 평일 09:00 ~ 15:45
- ✅ **야간장 거래중**: 평일 18:00 ~ 익일 05:00
- 🔴 **장 마감**: 위 시간 외 모든 시간 (주말 포함)
