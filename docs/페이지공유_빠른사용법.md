# 🎯 페이지 공유 빠른 사용법

## 화면 상단에 추가된 4개 버튼

대시보드 페이지를 열면 **화면 상단 헤더**에 4개의 파란색 버튼이 있습니다:

```
선물/옵션 실시간 거래 대시보드        [📋] [💾] [📄] [💻]  15:30:45  🟢 주간장 거래중
```

---

## ⚡ 가장 빠른 방법 (추천!)

### 1️⃣ 클립보드로 복사하여 공유

**방법 1:** 첫 번째 버튼 📋 클릭  
**방법 2:** 키보드에서 `Ctrl + Shift + C` 입력

→ "페이지 상태가 클립보드에 복사되었습니다!" 메시지 확인  
→ 메모장, 이메일, 카카오톡 등에 `Ctrl + V`로 붙여넣기

**이렇게 나옵니다:**
```
================================
선물/옵션 대시보드 스냅샷
================================

캡처 시간: 2025-12-20 15:30:45
시장 상태: 주간장 거래중
데이터 소스: KIS

--- 선물 전체 ---
거래량: 1,234,567
거래대금: 123.4억원
미결제약정: 456,789
...
```

---

## 📝 다른 방법들

### 2️⃣ 파일로 저장하여 공유

**방법 1:** 두 번째 버튼 💾 클릭  
**방법 2:** 키보드에서 `Ctrl + Shift + S` 입력

→ `dashboard-report-2025-12-20T15-30-45.txt` 파일이 다운로드됩니다  
→ 이 파일을 이메일에 첨부하거나 공유하면 됩니다

### 3️⃣ 스크린샷 캡처 (시각적으로 보여주기)

**Windows 기본 기능 사용:**
- `Windows + Shift + S` 키 입력
- 마우스로 원하는 영역 드래그
- 자동으로 클립보드에 복사됨
- `Ctrl + V`로 붙여넣기

### 4️⃣ 개발자용 JSON 파일

세 번째 버튼 📄 클릭 또는 `Ctrl + Shift + J`
→ 프로그램으로 분석 가능한 JSON 형식

## 🎬 실제 사용 시나리오

### 시나리오 1: 동료에게 카톡으로 현황 공유
1. `Ctrl + Shift + C` (클립보드 복사)
2. 카카오톡 열기
3. `Ctrl + V` (붙여넣기)
4. 전송! ✅

### 시나리오 2: 상사에게 이메일 보고
1. 💾 버튼 클릭 (파일 저장)
2. 이메일 작성
3. 방금 다운로드된 `.txt` 파일 첨부
4. 전송! ✅

### 시나리오 3: 회의 자료 준비
1. `Windows + Shift + S` (스크린샷)
2. PowerPoint에 `Ctrl + V` (붙여넣기)
3. 💾 버튼 클릭 (상세 데이터 파일도 준비)
4. 완료! ✅

---

## 💡 팁

- **가장 빠름:** `Ctrl + Shift + C` → `Ctrl + V`
- **파일 보관:** 💾 버튼으로 txt 파일 저장
- **시각 자료:** `Windows + Shift + S` 스크린샷
- **문제 발생 시:** F12 키로 콘솔 열어서 💻 버튼 클릭

## ❓ 자주 묻는 질문

**Q: 버튼을 눌렀는데 아무 일도 안 일어나요**  
A: 브라우저에서 다운로드를 차단했을 수 있습니다. 주소창 오른쪽의 팝업 차단 아이콘을 클릭하여 허용하세요.

**Q: 클립보드 복사가 안돼요**  
A: 브라우저 권한 설정에서 클립보드 접근을 허용해야 합니다. 또는 파일로 저장(💾) 후 내용을 복사하세요.

**Q: 파일이 어디에 저장되나요?**  
A: 브라우저 기본 다운로드 폴더에 저장됩니다 (보통 C:\Users\사용자명\Downloads)

**Q: 실시간으로 변하는 데이터도 캡처되나요?**  
A: 네! 버튼을 누른 그 순간의 데이터가 저장됩니다.

## 🎉 이제 사용하실 준비가 되었습니다!

1. 대시보드 페이지를 엽니다 → http://localhost:8080
2. 브라우저를 새로고침합니다 (F5)
3. 화면 상단에 4개의 파란색 버튼이 보입니다
4. 원하는 버튼을 클릭하거나 단축키를 사용하세요!

**가장 추천하는 방법:**
```
Ctrl + Shift + C → Ctrl + V
```
(클립보드 복사 후 바로 붙여넣기!)
