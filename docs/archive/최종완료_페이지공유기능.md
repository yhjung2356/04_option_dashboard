## 🚀 가장 빠른 사용법 (10초!)

```
1. 브라우저 새로고침 (F5)
2. Ctrl + Shift + C
3. 원하는 곳에 Ctrl + V
완료! 🎉
```

---

## 🎯 4가지 공유 방법

### 1️⃣ 📋 클립보드 복사 (가장 빠름!)
```
단축키: Ctrl + Shift + C
용도: 카톡, 이메일, 메모장 등 어디든
```

### 2️⃣ 💾 텍스트 파일 저장
```
단축키: Ctrl + Shift + S
파일명: dashboard-report-2025-12-20-153045.txt
용도: 보고서, 문서화, 아카이빙
```

### 3️⃣ 📄 JSON 파일 저장
```
단축키: Ctrl + Shift + J
파일명: dashboard-snapshot-2025-12-20-153045.json
용도: 데이터 분석, 프로그래밍, Excel
```

### 4️⃣ 💻 브라우저 콘솔 출력
```
방법: 버튼 클릭
용도: 디버깅, 실시간 확인
```

## 🎨 UI 변화

브라우저를 새로고침하면 **화면 상단**에 파란색 버튼 4개가 나타납니다:

```
┌─────────────────────────────────────────────────────────────────┐
│ 📈 선물/옵션 실시간 거래 대시보드                              │
│                                                                  │
│                    [📋] [💾] [📄] [💻]  15:30:45  🟢 주간장 거래중│
│                     ↑   ↑   ↑   ↑                              │
│                 클립보드|텍스트|JSON|콘솔                        │
└─────────────────────────────────────────────────────────────────┘
```

- **마우스 올리면**: 버튼이 살짝 위로 떠오름 ✨
- **클릭하면**: 우측 상단에 알림 표시 🔔
- **단축키**: 키보드로 빠르게 사용 ⚡

```
Ctrl + Shift + H
```

### 자동 스냅샷
```javascript
// 콘솔에서 실행: 5분마다 자동 스냅샷

    PageSnapshot.printToConsole();
```

```javascript
// 상태 변경 시 자동 알림
```

## 💬 자주 묻는 질문

**Q: 버튼이 안 보여요?**  
A: `F5` 키로 브라우저 새로고침!

**Q: 클립보드 복사가 안돼요?**  
A: 버튼 클릭 시 자동으로 폴백 방식 사용됨

**Q: 데이터가 0으로 나와요?**  
A: WebSocket 연결 확인, 시장 시간 확인, 잠시 대기

**Q: 파일이 어디에 저장되나요?**  
A: 브라우저 기본 다운로드 폴더 (보통 `다운로드` 폴더)
